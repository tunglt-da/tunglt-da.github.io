# Le Thanh Tung — Game Data Analyst Portfolio

Personal portfolio website built with HTML, CSS, Bootstrap 5, and vanilla JavaScript.

## Structure

```
portfolio/
├── index.html        # Home page (hero + highlights)
├── about.html        # About Me + Why Game Analytics
├── projects.html     # Featured projects with modals
├── skills.html       # Technical skills with progress bars
├── experience.html   # Work experience timeline
├── contact.html      # Contact form + info cards
├── style.css         # Shared styles
├── main.js           # Shared JavaScript
└── README.md
```

## Deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `tunglt-da.github.io` for a user site, or any name for a project site)
2. Upload all files to the repository root
3. Go to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Choose `main` branch and `/ (root)` folder, then click **Save**
6. Your site will be live at `https://tunglt-da.github.io/` (or `https://tunglt-da.github.io/repo-name/` for project sites)

## Features

- Cyberpunk / gaming aesthetic with Orbitron + Rajdhani fonts
- Responsive design (Bootstrap 5)
- Contact form powered by Google Apps Script
- Resume download via Google Drive
- Project detail modals
- Smooth navigation between pages
